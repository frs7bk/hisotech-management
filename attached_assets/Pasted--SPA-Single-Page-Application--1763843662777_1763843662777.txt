نظرة عامة على المشروع

مطلوب تطوير تطبيق ويب متكامل (SPA – Single-Page Application) لإدارة اشتراكات العملاء في منتجات متعددة، مع نظام مالي متقدم، ونظام لإدارة الحسابات الرئيسية، ومساعد ذكي قادر على تنفيذ جميع المهام داخل الموقع.
يجب أن يكون التطبيق جاهزًا للاستخدام مباشرة بعد التطوير، بواجهة عربية بالكامل ودعم RTL وتصميم نظيف مستوحى من أسلوب Apple: بسيط، سلس، مساحات واسعة، ألوان محايدة، وحركات خفيفة.

المتطلبات التقنية الأساسية

React 18+ مع TypeScript

TailwindCSS مع دعم الوضع المظلم

React Context API + Hooks لإدارة الحالة

Recharts للرسوم البيانية

localStorage كقاعدة بيانات أولية مع إمكانية الربط مستقبلاً بـ Backend

نظام ذكاء اصطناعي داخلي بدون اسم أو مزود محدد

يستطيع فهم اللغة العربية الطبيعية

يمتلك صلاحيات كاملة داخل النظام

يستطيع تنفيذ جميع العمليات: إضافات، تعديلات، بحث، إدارة مالية، إنشاء تقارير… إلخ

Web Speech API للأوامر الصوتية باللغة العربية

تصميم الواجهة (UI/UX – أسلوب Apple)

تصميم نظيف وهادئ وبسيط مع تأثيرات لطيفة

تباعد مريح بين العناصر

ألوان محايدة + Accent Color واحد

حركة Smooth عند التنقل، النوافذ المنبثقة، تحميل البيانات

شريط جانبي بسيط للتنقل

شريط علوي يضم: بحث سريع، الإشعارات، إعدادات المستخدم، تبديل الثيم

تجاوبي بالكامل على الهاتف والتابلت

1. إدارة المنتجات (Products Management)

لأن النظام يدعم بيع أكثر من منتج، وكل منتج يملك حساباته الرئيسية واشتراكاته.

المميزات:

إضافة/تعديل/تعطيل المنتجات

ربط كل منتج بحساباته الرئيسية الخاصة

عرض:

اسم المنتج

وصف مختصر

السعر القياسي

نوع الخطة (شهري/سنوي)

عدد الحسابات الرئيسية

عدد الاشتراكات الحالية

حالة المنتج

2. إدارة الحسابات الرئيسية (Master Accounts)

لكل منتج مجموعة حسابات رئيسية يتم الاشتراك من خلالها.

وظائف النظام:

إضافة أو تعديل حساب رئيسي

تحديد السعة القصوى لكل حساب

حساب الاستخدام الحالي ونسبته

إشعارات تلقائية عند:

الوصول إلى 80% من السعة

الوصول إلى 95%

عرض حالة الحساب: طبيعي – شبه ممتلئ – ممتلئ

إمكانية إيقاف حساب رئيسي عن استقبال اشتراكات جديدة

5. المساعد الذكي المتكامل في النظام

يفهم اللغة العربية الطبيعية بدون كلمات مفتاحية

يمتلك صلاحية كاملة لتنفيذ:

إضافة/تعديل/حذف اشتراكات

إدارة الإيرادات والمصاريف

إنشاء التقارير

البحث المتقدم

تنفيذ قواعد مالية

تغيير بيانات العملاء

إدارة المنتجات والحسابات الرئيسية

يتفاعل بالصوت أو النص

يعرض "معاينة" قبل تنفيذ الأوامر الحرجة مثل الحذف

يتعلم من سياق البيانات داخل الموقع

يستطيع تحليل البيانات المالية أو بيانات الاشتراكات مباشرة

مثال:
"اعرض لي اشتراكات منتج المحترف التي ستنتهي خلال أسبوعين"
"أضف مصروف تسويقي بقيمة 300 ريال"
"أنشئ تقرير أرباح لمنتج بريميوم"

بدون أي كلمات ثابتة — مجرد فهم طبيعي.
